# TouchFishRead(摸鱼阅读)
 摸鱼看书，摸鱼看小说必备
使用说明:
1. 请把要阅读的文本放到此目录
2. 用cmd进入此目录
3. 输入命令`./touchfish README.md`
4. 本目录会根据阅读文件名记录观看进度，记录在libfile文件，如果删除则会清除阅读记录
5. 如果懂lua可以打开script文件进行二次开发拓展命令
6. 使用过程中随时按`h`获取命令列表
